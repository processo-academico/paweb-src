Role: Implementador
O Implementador é responsável por desenvolver e testar componentes de acordo com os padrões adotados para o projeto, para fins de integração com subsistemas maiores.
Relationships
Performs:
  • Executar Builds Necessários e criar um Release
  • Executar Testes em Build Estável no Ambiente de Desenvolvimento
  • Implementar Correção de Erros
  • Verificar se Necessita passar por Implementação ou se deve passar por Remodelagem
Responsible for:
  • Códigos Fontes
Additionally Performs:
  • Planejar Testes em Ambiente de Desenvolvimento
Modifies:
  • Códigos Fontes
Properties
Multiple Occurrences
Optional
Planned
Staffing
Skills

As habilidades e conhecimentos apropriados para o implementador incluem:

  • Conhecimento do sistema no qual está trabalhando
  • Habilidades de programação
  • Conhecimento da linguagem de programação utilizada
  • Familiaridade com as ferramentas de programação e de testes
