Role: Analista
O Analista lidera e coordena a identificação de requisitos e a modelagem de casos de uso, delimitando o sistema e definindo sua funcionalidade, estabelecendo, por exemplo, quais são os atores e casos de uso existentes e como eles interagem.
Role Sets: Análise
Relationships
Primary Performs
  • Atualizar Diagrama de Casos de Uso e de Classes
  • Definir Escopo do Problema
  • Definir Funcionalidades e as Restrições do Sistema
  • Descrever Brevemente os Casos de Uso
  • Detalhamento dos Casos de Usos
  • Dividir o Problema em Casos de Uso
  • Elaborar Diagrama de Interação de Objetos
  • Escolher caso de uso para validar a tecnologia
  • Escolher Casos de Uso para ser detalhado
  • Identificar Mudanças nos Requisitos
  • Iniciar Alterações de Modelagem em Disciplina de Análise e Projeto
  • Planejar Testes em Ambiente de Desenvolvimento
  • Realizara Disciplina de Análise e Projeto da Fase de Contrução
  • Realizar Analise de Evolução do Projeto
  • Realizar Análise e Projeto
  • Realizar Comemoração de Fim de Projeto
Additionally Performs
  • Cliente Define o Projeto a ser Desenvolvido
  • Desenvolver Plano de Iteração
  • Escolher caso de uso para ser implementado na iteração
  • Executar Builds Necessários e criar um Release
  • Executar Implementação do Projeto
  • Executar Reuniões com equipe
  • Executar Testes em Build Estável no Ambiente de Desenvolvimento
  • Implementação do Caso de Uso Escolhido
  • Implementar Correção de Erros
  • Realizar a Disciplina de Implementação para o Caso de Uso Selecionado
  • Verificar se Necessita passar por Implementação ou se deve passar por Remodelagem
Modifies
  • Diagrama de Classes de Projeto (UML)
  • Diagrama de Interação de Objetos (UML) para o (s) Caso (s) de Uso (s) Implementado (s)
  • Diagrama Geral de Casos de Uso (UML)
  • Documento de Visão
  • Especificação de Caso de Uso

    Staffing
    Skills

    Um Analista precisa possuir os seguintes conhecimentos e habilidades:

    • Perícia em identificar e conhecer problemas e oportunidades
    • Habilidade em articular as necessidades associadas aos principais problemas a serem solucionados
    • Habilidade para colaborar de forma eficaz com a equipe em sessões de trabalho colaborativo, workshops e outras técnicas de trabalho
    • Boa habilidade de comunicação verbal ou escrita
    • Conhecimento acerca dos domínios do negócio e da tecnologia utilizada ou habilidade de absorver e compreender de forma rápida tais informações
