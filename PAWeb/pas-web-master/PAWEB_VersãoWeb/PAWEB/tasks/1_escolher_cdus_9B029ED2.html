Task: Escolher Casos de Uso para ser detalhado
São escolhidos os casos de uso que poderão ser desenvolvidos para validar a tecnologia candidata. Essa escolha é feita levando-se em consideração a complexidade e sua abragência. Mais de uma caso de uso é escolhido, pois o seu detalhamento é quem vai definir qual deles é o mais adequado para realizar a validação.
